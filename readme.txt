Данный модуль интеграции с Rficb разработан для Drupal-commerce 7.x-1.0.

Перед установкой модуля необходимо зарегистрироваться на сайте rficb.ru (раздел "Подключиться") и в панели управления магазином заполнить следующие поля, значения которых используются при настройке модуля:		
a) Секретный ключ - произвольный набор символов, используется для подписи сообщений при переадресации;	
b) Ключ - произвольный набор символов, используется для подписи скрытых сообщений на Result URL;	
c) Ключ API - произвольный набор символов, используется в случае дальнейшего расширения функций модуля разработчиками проекта.

Порядок установки модуля:
1. Скопируйте папку Modules в корневой каталог вашего сайта;
2. Зайдите в панель управления вашего сайта;
3. Откройте страницу настройки модулей (Administer/Site building/Modules);
4. Включите модуль Rficb - payment;
5. Перейдите на страницу настройки способов оплаты (Administer/Store administration/Configuration/Payment settings);
6. Нажмите кнопку "Редактировать" (Edit) и перейдите во вкладку "Payment methods";
7. Настройте способ оплаты, заполнив все необходимые поля:
 - Merchant ID: идентификатор вашего магазина в системе Rficb, указан в настройках вашего магазина в панели управления(вкладка Инструменты\Сервисы) на сайте Rficb;
 - Secret Key: секретный ключ, указан в настройках вашего магазина в панели управления на сайте Rficb(можно получить нажав кнопку в форме ключа напротив названия сервиса);
 - Hidden Key: Ключ, указан в настройках вашего магазина в панели управления на сайте Rficb(можно получить нажав кнопку в форме ключа напротив названия сервиса);
 - Test mode: установите тестовый режим работы;
6. Скопируйте в настройки магазина на сайте rficb.ru данные из следующих полей:
 - Success URL - страница, на которую будет перенаправлен плательщик после оплаты(страница успешной покупки в настройках сервиса в ЛК РФИ);
 - Fail URL - страница, на которую будет перенаправлен плательщик при отказе от оплаты(страница неуспешной покупки в настройках сервиса в ЛК РФИ);
 - Result URL - на этот адрес будет отправляться уведомление о состоянии платежа(URL скрипта обработчикав настройках сервиса в ЛК РФИ);
7. Сохраните изменения.

Для проверки настройки системы необходимо провести тестовые платежи, денежные средства по которым сниматься не будут. В панели управления магазином на сайте rficb.ru статус успешно выполненных операций будет иметь значение "Выполнен". 
Переключение в рабочий режим происходит после согласования с менеджером.
